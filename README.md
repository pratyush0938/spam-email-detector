# AI Spam Email Detector

A full-stack AI-powered spam email detection system built with:

- React + Vite frontend
- Node.js + Express backend
- Flask ML service
- MongoDB database
- Scikit-learn based spam classification
- Sender email risk analysis
- 3-level output: `spam`, `suspicious`, `not spam`

---

## Features

- User registration and login
- Protected dashboard
- Spam email detection using ML
- Sender email trust/risk analysis
- Suspicious category for borderline emails
- Prediction history
- Stats cards and chart visualization
- Premium animated UI
